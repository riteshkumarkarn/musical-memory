# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x   | :white_check_mark: |
| < 1.x   | :x:                |

## Reporting a Vulnerability

You can open an ![issue](https://github.com/warrior-guys/musical-memory/issues) to report any vulnerability, and we will look into it as soon as possible.
